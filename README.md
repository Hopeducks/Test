# 과학 마스터 메타버스 (Science Master Metaverse)

과학 마스터 메타버스는 초·중등 학생들이 메타버스 공간에서 아바타로 활동하며 실시간으로 퀴즈를 풀고, 희귀 과학 카드를 수집하며, 친구들과 카드 배틀 및 보스 레이드를 즐길 수 있는 실시간 과학 복습 플랫폼입니다.

---

## 💻 시스템 및 브라우저 요구사항
- **권장 브라우저**: **Google Chrome 110+** (Firefox 115+, Safari 16+, Edge 110+ 호환)
- **네트워크**: Supabase Realtime 연결을 위해 웹소켓(WebSocket) 통신이 차단되지 않은 환경이 필요합니다.
- **성능 및 규모**:
  - 한 교실 세션당 **동시 접속 40~60명**의 다중 접속 최적화 설계.
  - Supabase Free tier 요금제 기준 최대 **동시 연결 200개**까지 완벽 지원합니다.

---

## 👩‍🏫 교사 사용 매뉴얼 (Teacher Manual)

1. **교사용 대기실 개설**:
   - 브라우저로 `/teacher/create` 경로에 접속합니다.
   - 개설할 학습 단원(1~8단원)을 선택하고 **[대기실 로비 개설 개시]** 버튼을 클릭합니다.
   - 개설이 완료되면 자동으로 교사용 보안 접속키가 브라우저에 다운로드(JSON 파일)되며, 대시보드로 이동합니다.

2. **학생 참여**:
   - 대시보드 상단에 표시되는 대형 **참여 코드(예: ABC123)**를 칠판에 적어 학생들에게 공유합니다.
   - 학생들이 접속하면서 좌측 **대원 접속 그리드**에 아바타와 닉네임이 실시간으로 활성화되는지 확인합니다.

3. **실시간 콘텐츠 제어**:
   - **퀴즈 제어**: 우측 패널에서 단원을 선택하고 **[퀴즈 시작]**을 누르면 모든 학생의 화면이 메타버스에서 퀴즈 풀이 화면으로 강제 이동됩니다. 퀴즈가 끝나면 **[퀴즈 종료]**로 대기 로비로 복귀시킵니다.
   - **배틀 제어**: **[배틀 모드 ON]**을 활성화하면 아레나 구역이 열립니다. 교사는 미매칭 대원들을 마우스로 드래그하여 수동으로 1:1 매칭하거나 **[자동 매칭]**을 눌러 일괄 배틀을 시작할 수 있습니다.
   - **보스 레이드**: 학생들이 포탈에 진입하면 **[보스 레이드 활성화]**를 켭니다. 전체 학생들의 실시간 누적 데미지와 레이드 진행 상황(HP바)을 실시간 모니터링할 수 있습니다.

4. **학습 결과 관리**:
   - 수업 종료 후 **학습 통계** 탭에서 학생별 단원 클리어 현황 히트맵 및 평균 정답률을 시각적으로 확인합니다.
   - **[CSV 엑셀 내보내기]** 버튼을 클릭하여 전체 퀴즈 정오답 결과 데이터를 엑셀(CSV)로 내려받습니다.

## 🧑‍🎓 학생 접속 방법 (Student Manual)

1. **메타버스 로비 입장**:
   - 교사가 알려준 메타버스 URL에 접속합니다 (메인 페이지 `/`).
   - 사용하고자 하는 **닉네임**을 입력하고 세션 **참여 코드 6자리**를 입력합니다.
   - 중복 닉네임 검사가 자동으로 수행되며, 확인되면 입장 승인됩니다.

2. **아바타 커스터마이징 (옷장)**:
   - 로비 입장 전에 옷장 공간이 열립니다.
   - 피부색을 커스텀하고, 의상 | 악세서리 | 탈것 탭을 이동하며 장비를 변경합니다. 장착 시 좌측 아바타 캔버스에서 다이내믹하게 레이어 적층 및 둥실둥실 애니메이션이 적용됩니다.
   - 잠겨 있는 특수 아이템은 업적을 달성하거나 퀴즈 보상 코인을 사용하여 해금할 수 있습니다.
   - 준비를 마치면 **[로비 입장하기]** 버튼을 누릅니다.

3. **월드 탐험 및 미니게임**:
   - WASD 키 또는 방향키로 2D 메타버스 맵을 자유롭게 돌아다닙니다.
   - 교사의 제어에 의해 퀴즈/레이드가 시작되면 팝업을 통해 즉각 복습 활동에 참여합니다.
   - **희귀 카드 포획**: 퀴즈 정답 시 출현하는 움직이는 게이지 조준 바의 한가운데(녹색 바)를 타이밍 맞춰 터치하여 카드를 획득합니다.
   - **카드 배틀**: 아레나에 입장하여 매칭 완료 후, 보유한 카드 중 3장을 세팅해 1:1로 퀴즈 3판 2선승제 복습 대결을 치릅니다.

## 🎨 Phaser 4 에셋 교체 가이드 (픽셀아트 스프라이트 추가 방법)

본 플랫폼은 Phaser 4 기반의 그래픽 캔버스를 탑재하고 있습니다. 캐릭터 의상, 탈것, 이펙트 등의 픽셀 아트 스프라이트를 추가하려면 아래 가이드를 따르세요.

1. **스프라이트시트 준비**:
   - 애니메이션으로 동작할 캐릭터/아이템의 프레임별 스프라이트 시트를 `32x32` 또는 `64x64` 규격의 PNG 파일로 제작합니다.

2. **파일 저장**:
   - 제작된 PNG 에셋 파일을 `/public/assets/images/` 경로에 저장합니다.
   - 예: 새로운 전설 코스튬인 우주복을 추가할 경우 `/public/assets/images/costume_astronaut.png`

3. **Phaser preload 등록**:
   - `game/scenes/LobbyScene.ts` (또는 `PreloadScene`) 내 `preload()` 메소드에서 파일 키를 추가합니다.
   ```typescript
   this.load.spritesheet('costume_astronaut', '/assets/images/costume_astronaut.png', {
     frameWidth: 32,
     frameHeight: 32
   });
   ```

4. **아이템 메타데이터 등록**:
   - `data/costumes.ts` (또는 코스튬 카탈로그 파일)에 정해진 스키마 규격으로 새 코스튬 정보를 기입합니다.
   ```typescript
   export const COSTUME_CATALOG: CostumeItem[] = [
     // ...기존 아이템들
     {
       id: 'outfit_astronaut',
       name: '우주 비행복',
       category: 'outfit',
       rarity: 'legendary',
       unlockCondition: { type: 'achievement', achievementId: 'ach_space_master' },
       price: 150,
       spriteKey: 'costume_astronaut'
     }
   ];
   ```
   - 정상 등록 시, **아바타 커스터마이저** 샵 화면에 해당 우주 비행복 아이템이 자동으로 로드되고 해금 로직이 연동됩니다.
